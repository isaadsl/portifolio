# Portfólio

Este repositório contém meu portfólio pessoal desenvolvido com HTML, CSS e JavaScript.  
O objetivo é reunir meus projetos, certificados e experiências de forma organizada e acessível.

## Funcionalidades
- Página inicial com apresentação pessoal
- Seção de certificados com links para visualização
- Projetos desenvolvidos em HTML, CSS e JS
- Layout responsivo para diferentes dispositivos

## Estrutura do projeto
- `index.html` → Página principal
- `style.css` → Estilos do site
- `script.js` → Funcionalidades em JavaScript
- `Certificados/` → Certificados em PDF
- `portfolio/` → Subpasta com conteúdo adicional

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## Como visualizar
Você pode clonar este repositório e abrir o arquivo `index.html` em qualquer navegador.  
Se preferir, também pode acessar a versão publicada pelo GitHub Pages. (https://isaadsl.github.io/portifolio/)
